# ADVENT OF CODE 2024

This is my attempt to solve the Advent of Code 2024 using TypeScript.

## Environment

- nodes v22.11.0

## Setup

```bash
npm install
```

## Test

I'm using `jest` for testing.
Personally, I run the test with a vscode extension called `Jest Runner`.

OR run with specific test.

```bash
npm test day_00
```
